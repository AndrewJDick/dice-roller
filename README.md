# Lurch

A caretaker bot for the Discord application.

## Commands:

**roll:**\
Roll either a single die (e.g. **d20**) or multiple (e.g. **3d6**).\
`!lurch roll 2d8`

**help:**\
A list of all available commands.\
`!lurch help`

**about:**\
General information about Lurch.\
`!lurch about`

**ping:**\
Ping Lurch's server.\
`!lurch ping`
